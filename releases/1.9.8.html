What is the Microbot Hub?

The Microbot Hub is a dedicated, separate repository designed specifically for community-created plugins and scripts. Think of it as the Grand Exchange for Microbot plugins - a centralized marketplace where developers can share, discover, and collaborate on custom automation solutions.

Key Purpose: Prevent bloating of the main Microbot client while fostering rapid community innovation

Why Does the Microbot Hub Exist?

üéØ Core Design Philosophy

  • Lean Main Client: Keep the core Microbot client small, stable, and focused on essential functionality
  • Rapid Innovation: Enable community developers to create and share plugins without waiting for main client updates
  • Quality Control: Separate experimental/community plugins from production-ready core features
  • Scalability: Allow unlimited plugin growth without impacting main client performance
‚úÖ Result: A stable, fast core client with an unlimited ecosystem of community plugins

‚òï Java Version Mismatches

Problem: "Unsupported class file major version" error
Solution:
# Check your Java version java -version # Set JAVA_HOME to correct JDK export JAVA_HOME=/path/to/java11 # Verify Gradle is using correct Java ./gradlew -version

üîå Missing Plugin Dependencies

Problem: Plugin fails to load due to missing dependencies
Solution:
  • Check dependencies.txt in plugin folder
  • Install missing libraries: ./gradlew build --refresh-dependencies
  • Verify all required plugins are enabled
